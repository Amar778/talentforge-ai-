# TalentForge AI 🚀

> **Next-Generation AI-Powered Recruitment & Career Development Platform**

TalentForge AI is a modern web application designed to bridge the gap between job seekers and recruiters using artificial intelligence. It empowers candidates to build optimized resumes and practice interviews, while providing recruiters with intelligent candidate matching and automated job posting tools.

---

## ✨ Key Features

### 👨‍🎓 For Candidates
- **📄 ResumeForge**: AI-driven resume builder with real-time feedback, skill optimization, and formatting suggestions.
- **🎙️ Interview Simulator**: Realistic AI voice/text interview practice environment with detailed performance evaluation and tips.
- **🗺️ Skill Roadmap**: Personalized career path recommendation and skill gap visualization.
- **💼 Candidate Portal**: Unified dashboard to manage applications, saved jobs, and interview progress.

### 👔 For Recruiters
- **🎯 Candidate Matcher**: Intelligent matching algorithms that pair job requirements with top-fit candidate profiles.
- **📝 Job Post Studio**: AI-assisted job description generator to quickly craft engaging role specifications.
- **📊 Recruitment Analytics**: Comprehensive dashboard tracking hiring pipelines, candidate conversion, and metrics.
- **🏢 Recruiter Portal**: Central hub to manage open roles, candidate pipelines, and scheduling.

## 🚀 Getting Started

### Prerequisites

Ensure you have **Node.js** (v18 or higher) and **npm** installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/YOUR_USERNAME/talentforge-ai.git
   cd talentforge-ai
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open in browser:**
   Navigate to [http://localhost:3000](http://localhost:3000) to view the app.

---

## 📦 Scripts

- `npm run dev`: Starts the Next.js development server.
- `npm run build`: Compiles and builds the production bundle.
- `npm run start`: Starts the Next.js production server.
- `npm run lint`: Runs ESLint for code analysis.

---

## 🌐 Deployment

The easiest way to deploy TalentForge AI is with **Vercel**:

1. Push your code to GitHub.
2. Import your repository into [Vercel](https://vercel.com/).
3. Vercel will automatically detect Next.js and deploy your application.
